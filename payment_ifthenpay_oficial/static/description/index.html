ifthenpay Payment Gateway for Odoo

Support for MULTIBANCO, MB WAY, Apple Pay, Google Pay, PIX, and much more

This module provides full integration between Odoo and the ifthenpay payment gateway, enabling fast and secure payments directly from your Odoo website.

Key Features
  • Generate MULTIBANCO payment references automatically
  • Receive MB WAY payments in real time
  • Enable Apple Pay and Google Pay in checkout
  • PIX, Credit Card and Payshop support
Free Service Subscription
  • Visit the website ifthenpay.com
  • Download and fill out the contract.
  • Attach the requested documents.
  • Request the creation of the Gateway Key.
  • Send the documents to ifthenpay@ifthenpay.com
  • Note: If you already have a contract with ifthenpay, simply request the creation of the Gateway Key.

For more information, visit our website at ifthenpay.com.

Easy Setup

After installing the module from the "apps" page, you can configure the ifthenpay provider in the Odoo admin panel using your ifthenpay account credentials. You can activate one or more payment methods (through the ifthenpay back office) and start accepting payments immediately in your Odoo.
